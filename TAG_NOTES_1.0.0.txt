## Language Weighting (New!)

Customize which languages are prioritized:
  ‚Ä¢ weight: 1.0 (default) - normal detection
  ‚Ä¢ weight: 0.5 - deprioritize (e.g., Markdown at 0.25)
  ‚Ä¢ weight: 1.5+ - prioritize your main language

Example: Markdown is weighted at 0.25 to prevent large README files from incorrectly identifying the language.

## Installation

Choose your method:

1. Direct: chmod +x detect_repo_language.py && ./detect_repo_language.py --primary-only /path/to/repo
2. Wheel: pip install dist/detect_repo_language-1.0.0-py3-none-any.whl && detect-repo-language --primary-only /path/to/repo
3. Arch:  makepkg -si && detect-repo-language --primary-only /path/to/repo
